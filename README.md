## Installation

You can install the package via composer:

```bash
composer require aymanalhattami/unit-conversion
```

## Usage

```php
$skeleton = new Aymanalhattami\UnitConversion();
echo $skeleton->echoPhrase('Hello, Aymanalhattami!');
```
